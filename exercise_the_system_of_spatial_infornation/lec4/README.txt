# README

This folder contains the materials for the #5 lecture.

Please download all the files in this folder to your own Google Drive.

Contents:

* 00_clean_GPS_data: In this notebook, we will clean the GPS data of one user. You may skip this notebook; this is a notebook for essentially data preparation.

* 01_feature_extraction.ipynb: In this notebook, we will extract some features from GPS data.

* 02_machine_learning.ipynb: In this notebook, we will explore some basic machine learning models to predict transportation modes with the extracted features.

* 03_hw1.ipynb: This notebook is for homework (see ../#6_human_mobility_analysis_3/instruction_hw.gdoc for more instruction).
